Сервис "рублевый Биткоин". ОДИН МОДУЛЬ
Необходимо создать Spring web приложение, которое
1. Будет по адресу /rate/now отдавать стоимость биткоина в рублях.
Стоимость биткоина в долларах можно получить по этому url
https://api.coindesk.com/v1/bpi/currentprice.json
Получить стоимость доллара можно здесь https://app.freecurrencyapi.com
(предварительно нужно зарегистрироваться)
Все успешные результаты запросов нужно сохранять
(т.е хранить стоимость биткоина в рублях + время и дату)
2. По адресу /rate/history отдавать результаты всех успешных запросов (из пункта 1)
3. По адресу /rate/average отдавать среднюю стоимость биткоина исходя из всех успешных запросов
(из пункта 1). Если ничего нет, выкидывать RuntimeException("No rates now!").
4. В случае ошибок на сервере, пользователь должен получить ответ
"На сайте наблюдаются проблемы, приходите позже".
Например если из пункта 3 вылетит ошибка, она должна быть перехвачена.
